# MHST3 Tool — 魔物獵人 Stories 3 查詢工具

一個給 Monster Hunter Stories 3 玩家用的網頁小工具，可以查魔物資料、看弱點、規劃基因配置。

目前收錄了超過 130 隻魔物，包含一般魔物、天變古龍、兇異魔物跟一些特殊魔物。

## 有什麼功能？

**魔物圖鑑**

可以瀏覽所有魔物的清單，支援用名稱搜尋（中文、英文、日文都可以），也能用種族、攻擊類型、分類來篩選。點進去會看到每隻魔物的詳細資料：

- 攻擊類型（力量 / 速度 / 技巧）和特殊攻擊的條件切換
- 屬性弱點（火、水、雷、氷、龍）
- 騎乘能力（飛行、攀爬、游泳等等）
- 出現地點跟取得方式
- 羈絆基因列表，還能展開看哪些魔物也有同樣的基因

**基因規劃器**

一個 3×3 的基因格配置工具，可以：

- 從基因清單裡挑選基因放到格子裡
- 查詢每個基因被哪些魔物持有
- 瀏覽所有魔物的基因一覽表

## 技術

- React 19 + TypeScript
- Vite
- React Router (Hash Router)
- 搜尋支援拼音（用了 pinyin-pro）

純前端，沒有後端，資料都寫在程式碼裡面。

## 怎麼跑起來？

先確定你有裝 Node.js，然後：

```bash
cd web
npm install
npm run dev
```

打開瀏覽器就可以用了。

要 build 的話：

```bash
npm run build
```

產出的檔案會在 `web/dist/` 裡面。

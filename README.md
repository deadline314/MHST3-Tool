# MHST3 Tool — 魔物獵人 Stories 3 查詢工具

一個給 Monster Hunter Stories 3 玩家用的網頁小工具，可以查魔物資料、看弱點、規劃基因配置，還能查武器跟防具。

目前收錄了超過 130 隻魔物、200+ 把武器、80+ 件防具，包含一般魔物、天變古龍、凶異魔物跟一些特殊魔物。

## 有什麼功能？

**魔物圖鑑**

可以瀏覽所有魔物的清單，支援用名稱搜尋（中文、英文、日文都可以，也支援語音輸入），還能用很多條件來篩選跟排序。點進去會看到每隻魔物的詳細資料：

- 攻擊類型（力量 / 速度 / 技巧）和特殊攻擊的條件切換
- 屬性弱點（火、水、雷、氷、龍）
- 異常狀態有效性（毒、麻痺、睡眠、暈眩等）
- 傷害效果（斬擊 / 打擊 / 穿刺對各部位的有效度，多型態的還能切換）
- 騎乘能力（飛行、攀爬、游泳等等）
- 出現地點跟取得方式
- 賓果數加成（2 / 3 / 5 賓果的效果）
- 隨行獸能力傾向（生命力、攻擊力、會心率、速度、防禦力、耐力回復、耐力初始）
- 羈絆基因列表，還能展開看哪些魔物也有同樣的基因
- 相關魔物（同系列的亞種、特殊種可以快速跳轉）

**篩選與排序**

列表頁的篩選功能大幅強化：

- 依地區篩選（大區域 + 可展開子地點）
- 依騎乘能力篩選（飛行、攀爬、游泳、潛地、隱密等）
- 依騎乘攻擊屬性篩選（火、水、雷、冰、龍、無）
- 多種排序方式（預設、中文名、英文名、種類、地區、隨行獸能力傾向），排序方向可切換

**基因規劃器**

一個 3×3 的基因格配置工具，可以：

- 從基因清單裡挑選基因放到格子裡
- 放上基因後，下方查詢列表會自動篩選成九宮格上有用到的基因
- 支援基因名稱跟效果文字搜尋
- 依攻擊類型、屬性、技能類型、對象、異常狀態來篩選基因
- 查詢每個基因被哪些魔物持有
- 瀏覽所有魔物的基因一覽表

**裝備檢索**

可以瀏覽所有武器跟防具的資料：

- 武器：大劍、太刀、大鎚、狩獵笛、弓、銃槍，共 200+ 把，每把都有攻擊力、屬性、會心率、破龍力、狀態異常、插槽等數值
- 防具：80+ 件，有防禦力、六屬性耐性、插槽、技能、素材需求
- 支援武器種、屬性、狀態異常、插槽數等多條件篩選，還能搜素材跟搜技能
- 點進武器可以看完整資訊：各等級攻擊力、武器技能（中日對照）、強化素材、素材細節，素材還會連結到對應的魔物頁面
- 武器名稱跟技能都有日文→繁體中文翻譯

## 技術

- React 19 + TypeScript
- Vite
- React Router (Hash Router)
- 搜尋支援拼音（用了 pinyin-pro）
- 支援 Web Speech API 語音搜尋

純前端，沒有後端，資料都寫在程式碼裡面。

## 怎麼跑起來？

先確定你有裝 Node.js，然後：

```bash
cd web
npm install
npm run dev
```

打開瀏覽器就可以用了。

要 build 的話：

```bash
npm run build
```

產出的檔案會在 `web/dist/` 裡面。

# 更新日誌

## v1.1.0（2026-03-19）

這次更新蠻大的，主要是把魔物詳細頁的資訊補得更完整，列表頁的篩選排序也大幅升級了，另外還加了語音搜尋。

### 新功能

**魔物詳細頁 — 新增好幾個資訊區塊**

- **異常狀態有效性**：現在每隻魔物的頁面都能看到牠對各種異常狀態的抗性了，包含毒、火傷、麻痺、睡眠、爆破、出血、目眩、暈眩、屬性封印、技能封印、龍捲風、即死，總共 12 種。用星星數量來表示有效程度，還有對應的小 icon，一眼就能看懂
- **傷害效果表（部位弱點）**：新增斬擊 / 打擊 / 穿刺對各部位的有效度，用星星（有效）跟叉叉（無效）顯示。像是有些魔物有多種型態（通常、憤怒、帶電等），會有 tab 可以切換看不同狀態下的部位弱點
- **賓果數加成**：顯示每隻魔物 2 賓果 / 3 賓果 / 5 賓果各自的加成效果（像是破龍力、會心率、速度、耐力等），規劃基因的時候很好用
- **相關魔物**：頁面底部會列出同系列的魔物（原種、亞種、二名、兇異等），有小圖示跟名稱，點了就能直接跳過去。總共建了 40 幾組魔物家族的對照關係

**列表頁篩選與排序大升級**

- **地區篩選**：可以用四大區域（亞茲萊爾領域、卡納爾塔密林、塔基爾坎、桑提亞）來篩選，還有「展開子地點」按鈕可以看到更細的 16 個子地點來做精確篩選
- **騎乘能力篩選**：新增 7 種騎乘能力的篩選 chip（飛行、攀爬、游泳、潛地、隱密、咆哮、吐息），可以多選，選了的話就只會顯示具備所有你選的能力的魔物
- **騎乘攻擊屬性篩選**：可以依騎乘攻擊的屬性來篩（火、水、雷、冰、龍、無），選中的 chip 會顯示對應的屬性顏色
- **排序功能**：新增排序下拉選單，有五種排序方式 — 預設排序、中文名稱、英文名稱、種類、地區

**語音搜尋**

- 搜尋欄旁邊多了一個麥克風按鈕，按下去就可以用語音來搜尋魔物名稱。用的是瀏覽器內建的 Web Speech API，語言設定為繁體中文（zh-TW），支援即時辨識（邊講邊顯示）。錄音中按鈕會變紅色並閃爍提示

**基因規劃器改善**

- 在九宮格放上基因之後，下方的基因查詢列表會自動篩選成只顯示你有用到的基因，不用再從一大堆基因裡面慢慢找了。提示文字也會跟著變

### 資料新增與修正

**新增資料**

- 新增 23 隻兇異魔物的出現地點與取得方式資料（兇異查加布魯、兇異彩鳥、兇異速龍王、兇異毒怪鳥、兇異泡狐龍、兇異大名鑽蟹……等等）
- 新增異常狀態有效性資料（statusEffectiveness.ts），涵蓋大部分魔物
- 新增傷害效果 / 部位弱點資料（damageEffectiveness.ts），含多型態資料
- 新增賓果加成資料（bingoBonus.ts），涵蓋大部分可騎乘魔物
- 新增全部 7 張異常狀態 icon 圖片（毒、火傷、麻痺、睡眠、爆破、出血、暈眩）
- 新增地區列表資料（4 大區域 + 16 個子地點），支援地區篩選用

**資料修正**

- 修正水獸（Ludroth）的特殊攻擊：從 `-` 改為 `[憤怒]: 技巧`
- 修正恐暴龍（Deviljho）的屬性弱點：從全部「不明」改為正確數值（冰大弱點、龍微耐性）
- 修正 Dos Poogie 的基因資料 key（從 `Dospoogie` 改成 `Dos Poogie`，跟其他地方統一）
- 修正 3 個基因翻譯名稱：蒼雷電壓→青電壓、蒼雷電漿→電漿光束、雷光刃→光刃斬
- 統一「危惧種巢穴」用語為「侵獸巢穴」（影響電龍、雷狼龍、迅龍、轟龍、冰牙龍、雌火龍、電龍等 8 隻魔物的取得方式描述）
- 暫時註解掉尚未確認完整資料的 4 隻古龍種（麒麟、炎王龍、鋼龍、滅盡龍），等資料補齊再放回來
- 新增兇異魔物的弱點對照表（Afflicted → Feral 對應），讓兇異魔物也能正確顯示屬性弱點

### 技術 / UI 改動

- **魔物 ID 格式改版**：從流水號 `monster-0`, `monster-1` 改成用英文名的 slug（例如 `rathalos`, `azure_rathalos`），網址更好看也更穩定，不會因為資料順序調整就壞掉
- **ScrollToTop 元件**：切換頁面時會自動捲回頂端，之前從詳細頁回列表頁會卡在奇怪的位置
- **TypeScript 型別補充**：加了 `BingoBonus` interface、`SpeechRecognition` 相關的 Web API 型別定義（在 vite-env.d.ts），還有 `SortOption`、`RideAbility` 等新型別
- **篩選面板 UI 重整**：移除了一些多餘的 `filter-header` 包裝 div，排版更乾淨。地區篩選的子地點有左邊線條的縮排效果
- **搜尋欄 padding 調整**：右邊空間加大，讓語音按鈕跟清除按鈕不會擠在一起
- **全新 CSS 樣式**：傷害效果表、異常狀態表、賓果加成、相關魔物卡片、排序選單、地區 chip、騎乘能力 chip、騎乘屬性 chip、語音按鈕動畫，全部都是新做的
- Dos Poogie 的 icon 圖片有更新過（檔案大小從 23KB → 356KB）

## v1.0.0

第一版上線！基本功能都有了：

- 魔物圖鑑：超過 130 隻魔物的清單，支援中 / 英 / 日搜尋，可用種族、攻擊類型、分類篩選
- 魔物詳細頁：攻擊類型、屬性弱點、騎乘能力、出現地點、羈絆基因
- 基因規劃器：3×3 九宮格配置，基因查詢與魔物對照
- 支援拼音搜尋
- 純前端架構，用 React 19 + Vite + TypeScript 打造

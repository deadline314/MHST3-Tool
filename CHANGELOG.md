# 更新日誌

## v1.1.0（2025-03-19）

這次更新主要是把魔物詳細頁的資訊補得更完整，然後列表頁的篩選功能也大幅升級了。

### 魔物詳細頁新增資料

- **異常狀態有效性**：現在可以看到每隻魔物對毒、麻痺、睡眠、暈眩、爆破、出血、火屬性異常的抗性了，用星星跟 icon 直覺顯示
- **傷害效果表**：新增斬擊 / 打擊 / 穿刺對各部位的有效度，有多型態的魔物（像是憤怒狀態）還可以切換 tab 來看不同狀態的數據
- **賓果數加成**：現在會顯示 2 / 3 / 5 賓果各自的加成效果
- **相關魔物區塊**：在頁面底部會列出同系列的魔物（亞種、特殊種等等），點了就能直接跳過去看

### 篩選與排序大升級

- **地區篩選**：可以用大區域（亞茲萊爾領域、卡納爾塔密林等）來篩選魔物，也可以展開看子地點做更細的篩選
- **騎乘能力篩選**：可以篩選有特定騎乘能力的魔物，像是飛行、攀爬、游泳、潛地、隱密、咆哮、吐息
- **騎乘攻擊屬性篩選**：可以依騎乘攻擊的屬性（火、水、雷、冰、龍、無）來篩
- **排序功能**：新增排序選項，可以用中文名、英文名、種類、地區來排序

### 搜尋功能強化

- **語音搜尋**：搜尋欄旁邊多了麥克風按鈕，支援用語音輸入來搜尋魔物名稱（用的是瀏覽器內建的 Web Speech API，設定為繁體中文）

### 基因規劃器改善

- **自動篩選基因列表**：在九宮格放上基因之後，下方的基因查詢列表會自動篩選成只顯示你有用到的基因，找資料更方便

### 資料修正

- 修正水獸（Ludroth）的特殊攻擊資訊
- 暫時移除尚未確認完整資料的部分古龍種（麒麟、炎王龍、鋼龍、滅盡龍）
- 魔物 ID 改用英文名稱的 slug 格式，網址更好看也更穩定

### 樣式與 UI

- 新增異常狀態 icon 圖片
- 傷害效果表、異常狀態表、賓果加成、相關魔物的樣式都是全新設計的
- 篩選面板的版面重新整理過，加了排序下拉選單
- 地區篩選支援展開 / 收合子地點

## v1.0.0

第一版上線！基本功能都有了：

- 魔物圖鑑：超過 130 隻魔物的清單，支援中 / 英 / 日搜尋，可用種族、攻擊類型、分類篩選
- 魔物詳細頁：攻擊類型、屬性弱點、騎乘能力、出現地點、羈絆基因
- 基因規劃器：3×3 九宮格配置，基因查詢與魔物對照
- 支援拼音搜尋
- 純前端架構，用 React 19 + Vite + TypeScript 打造

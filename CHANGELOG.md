# 更新日誌

## v1.5.0（2026-03-22）

這版最大的亮點是全新的「裝備檢索」系統，武器跟防具終於可以查了！另外也幫隨行獸加上了能力傾向數據，基因規劃器的篩選也更好用了。

### 新功能

**裝備檢索系統（全新頁面）**

- 導航列新增「裝備檢索」頁籤，現在有三個主要功能了：魔物圖鑑、基因規劃器、裝備檢索
- 武器資料收錄了 6 種武器類型（大劍、太刀、大鎚、狩獵笛、弓、銃槍），總共 200+ 把武器，每把都有攻擊力、屬性、會心率、破龍力、狀態異常、防禦加成、插槽等完整數值
- 防具資料收錄了 80+ 件防具，包含防禦力、六屬性耐性（無/火/水/雷/氷/龍）、插槽、技能、素材需求
- 武器卡片上直接顯示武器技能的 preview chip（主動技能跟被動技能用不同顏色標示）
- 支援多種篩選：武器種、屬性、狀態異常、最少插槽數，還有素材搜尋跟技能搜尋
- 排序功能：可以依攻擊力、會心率、破龍力、插槽數、名稱來排序，方向可切換

**武器詳細頁面**

- 點武器卡片可以進入獨立的武器詳細頁，有大大的武器類型 icon、屬性標籤、狀態異常標籤
- 基本數值區塊：攻擊力（含各等級數值）、會心率、破龍力、防禦力、插槽
- 武器技能區塊：每個技能都有獨立卡片，顯示技能名稱（中日對照）、主動/被動標籤、對象標籤，主動技能額外顯示耐力消耗、威力、破龍力
- 強化素材區塊：列出每個等級需要的素材跟 pts
- 素材細節區塊：展開每個素材分類下的具體素材項目跟 pts
- 素材名稱會自動連結到對應的魔物詳細頁（例如「火竜素材」→ 火龍的頁面）

**武器名稱 / 技能 / 素材翻譯系統**

- 新增了完整的武器相關日文→繁體中文翻譯（約 650 行），涵蓋武器技能名稱、技能效果、武器名稱、素材名稱
- 武器技能名稱翻譯：蓄力斬、氣刃斬、橫掃、流星大鎚、攻勢之歌、龍砲彈等等，全部都有對照
- 技能效果翻譯：把日文的效果說明轉成繁體中文，像是「相手全体に武器属性ダメージを与える」→「對敵方全體造成武器屬性傷害」
- 每把武器都有武器類型 icon（大劍、太刀、大鎚、狩獵笛、弓、銃槍各一張 WebP 圖）

**隨行獸能力傾向**

- 魔物詳細頁新增「能力傾向」區塊，顯示隨行獸的 7 項能力數值：生命力、攻擊力、會心率、速度、防禦力、耐力回復、耐力初始
- 用橫條圖（bar chart）呈現，不同數值區間用不同顏色（紅色 ≥8、橙色 ≥6、藍色 ≥4、灰色 <4），一眼就能看出強項跟弱項
- 收錄了大約 90 隻可騎乘魔物的能力傾向數據
- 篩選面板的「分類」新增「隨行獸」選項，可以快速篩出所有可騎乘的魔物

**排序功能升級**

- 魔物列表的排序現在多了 7 種隨行獸能力傾向排序（生命力、攻撃力、會心率、速度、防禦力、耐力回復、耐力初始）
- 排序方向可以切換了（遞增 / 遞減），旁邊有一個箭頭按鈕可以點
- 能力傾向排序預設為遞減（高的排前面），其他排序預設為遞增

**基因規劃器篩選強化**

- 基因查詢列表新增搜尋框，可以用基因名稱或效果文字來搜尋
- 基因篩選新增「異常狀態」分類，可以篩選跟特定異常狀態相關的基因（毒、灼傷、麻痺、睡眠、爆破、裂傷、目眩、暈眩、即死、技能封鎖）
- 異常狀態篩選是根據基因效果文字裡的關鍵字自動判斷的，像是「賦予毒」「無效化麻痺」這些

### 資料修正

**基因翻譯修正**

## v1.4.1（2026-03-20）

這次是效能優化版本，主要把圖片全面轉成 WebP 格式大幅減少載入大小，魔物列表也改用無限捲動避免一次渲染上百張卡片，另外加上 Vercel 靜態資源快取和 robots.txt。

### 效能優化

**圖片格式從 PNG 轉 WebP**

- 所有魔物 icon（130+ 張）新增 WebP 版本，程式碼全面改為載入 `.webp` 格式
- 部分狀態異常 icon（目眩、屬性封印、技能封印、龍捲風、即死）也新增 WebP 版本
- WebP 相比 PNG 檔案大小大幅縮減，頁面載入速度明顯提升

**魔物列表無限捲動（Infinite Scroll）**

- `MonsterGrid` 改用 `IntersectionObserver` 分批載入，初始只渲染 30 隻魔物卡片
- 當使用者滾到列表底部才自動載入下一批（每批 30 隻），避免一次渲染全部 130+ 張卡片造成卡頓
- 篩選條件變更時會自動重置為第一批，使用體驗不變

### 部署 / 設定

**Vercel 快取設定**

- 新增 `vercel.json`，針對魔物 icon、狀態異常 icon、icons.svg、logo.png 設定 `Cache-Control: public, max-age=31536000, immutable`（一年快取）
- favicon.svg 設定 30 天快取

**robots.txt**

- 新增 `robots.txt`，允許一般搜尋引擎爬蟲，阻擋 AI 訓練用爬蟲（GPTBot、CCBot、anthropic-ai、Google-Extended 等）

### 資料修正

- 刪除命名錯誤的 `MHST3-Feral_Rakna_Kadaki_Icon.png`（正確檔名為 `MHST3-Feral_Rakna-Kadaki_Icon`，Kadaki 前面是 `-` 不是 `_`）

## v1.4.0（2026-03-19）

這次更新最大的亮點是基因詳細資訊系統，點基因終於不是只看到「誰有這基因」了，現在可以直接看到技能的屬性、類型、威力、耐力消耗、效果說明等等完整資訊。另外這版也修了一大堆魔物的中文名稱跟特殊攻擊名稱，讓翻譯更貼近遊戲內的正式用語。

### 新功能

**基因詳細資訊彈窗**

- 在基因規劃器跟魔物詳細頁點基因，不再只是展開一個小下拉選單了，改成會跳出一個漂亮的 Modal 彈窗
- 彈窗裡面可以看到基因的完整資訊：攻擊類型（力量 / 速度 / 技巧）、屬性（火水雷冰龍 / 無屬性）、技能類型（主動 / 被動）、對象（敵單體 / 敵全體 / 自身等）
- 主動技能會額外顯示耐力消耗、威力、破龍值、習得等級這些數值
- 每個基因都有效果說明文字，像是「對敵單體造成火屬性傷害，機率賦予灼傷」這種
- 彈窗下方一樣會列出擁有這個基因的魔物清單，點了就能跳到魔物頁面
- 目前收錄了大約 80+ 個基因的詳細資料，還沒補到的會顯示「此基因的詳細資料尚待補充」

## v1.2.1（2026-03-19）

這次是小幅更新，主要是屬性弱點系統補上「無屬性」欄位、狀態異常 icon 補齊，還有超龍化蒼火龍跟超龍化冰狼龍的翻譯修正跟資料補完。

### 新功能

**屬性弱點新增「無屬性」**

- 原本屬性弱點只有火、水、雷、冰、龍五種，現在多了「無屬性」這一欄
- 魔物詳細頁的弱點表格會在最前面顯示無屬性的耐性（普通 / 弱點 / 大弱點 / 微耐 / 耐性 / 不明）

### 資料修正

**超龍化蒼火龍 / 超龍化冰狼龍 翻譯與資料修正**

- 束縛蒼火龍 → 超龍化蒼火龍，日文從 束縛リオレウス亜種 改為 超龍化リオレウス亜種

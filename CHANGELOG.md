# 更新日誌

## v1.2.1（2026-03-19）

這次是小幅更新，主要是屬性弱點系統補上「無屬性」欄位、狀態異常 icon 補齊，還有超龍化蒼火龍跟超龍化冰狼龍的翻譯修正跟資料補完。

### 新功能

**屬性弱點新增「無屬性」**

- 原本屬性弱點只有火、水、雷、冰、龍五種，現在多了「無屬性」這一欄
- 魔物詳細頁的弱點表格會在最前面顯示無屬性的耐性（普通 / 弱點 / 大弱點 / 微耐 / 耐性 / 不明）
- 無屬性的顏色用灰色顯示（#95a5a6），跟其他屬性區分開來

### 資料修正

**超龍化蒼火龍 / 超龍化冰狼龍 翻譯與資料修正**

- 束縛蒼火龍 → 超龍化蒼火龍，日文從 束縛リオレウス亜種 改為 超龍化リオレウス亜種
- 束縛冰狼龍 → 超龍化冰狼龍，日文從 束縛ルナガロン 改為 超龍化ルナガロン
- 補上兩隻的特殊攻擊：超龍化蒼火龍為 `[憤怒]: 速度 / [超龍化解放]: 技巧`，超龍化冰狼龍為 `[憤怒]: 力量 / [超龍化解放]: 技巧`
- 新增兩隻的屬性弱點資料（無屬性、火水雷冰龍六欄）
- 新增兩隻的狀態異常耐性資料（毒、火傷、麻痺、睡眠、爆破、出血、目眩等）
- 新增 icon override，讓兩隻魔物能正確顯示對應的 icon 圖

**弱點資料格式改版**

- `weakness.ts` 的資料結構從 5 欄改成 6 欄（無屬性 + 火水雷冰龍），順序也改成按英文名 A-Z 排序，之後維護比較好找
- 補上幾隻之前沒在弱點表裡的魔物：食草龍（Aptonoth）、雪獅子（Blango）、盾蟹（Ceanataur）、鐮鼬龍（Izuchi）、水獸（Ludroth）

**狀態異常 icon 補齊**

- 原本目眩、屬性封印、技能封印、龍捲風、即死這五種狀態沒有 icon，現在都補上了
- 新增 5 張 icon 圖片：UI-Blind.png、UI-Typeseal.png、UI-Skillseal.png、UI-Cyclone.png、UI-ko.png
- 狀態異常耐性表格的 12 種狀態現在都有對應的 icon 了

**其他**

- 暫時註解掉草食獸（Slagtoth），等資料確認後再放回

### 技術 / 工具

- **parse_weakness.py**：新增一個 Python 腳本，可以從 AppMedia 的魔物頁面內容解析出屬性耐性資料。輸入 AppMedia 的 HTML 或文字檔，會輸出 6 欄格式（無屬性、火、水、雷、冰、龍）的 TypeScript 資料，方便之後從 AppMedia 抓資料來更新 weakness.ts。腳本內建日文↔英文魔物名稱對照表，涵蓋一般魔物、兇異、侵獸、天變古龍、超龍化等，包含 超龍化リオレウス亜種、超龍化ルナガロン 的對照
- **ElementResist 型別**：`monster.ts` 的 `ElementResist` interface 新增 `neutral` 欄位
- **ELEMENT_COLORS**：新增「無」和「無屬性」的顏色常數

## v1.2.0（2026-03-19）

這次更新的重點是侵獸系統跟突然變異的完整攻略資訊、搜尋功能大幅升級（模糊搜尋、拼音首字母），還有一波命名統一跟資料修正。

### 新功能

**侵獸攻略系統**

- 在侵獸的魔物詳細頁新增了完整的「侵獸攻略」區塊，包含：出現地區、詳細出現地點（含夜間限定等條件）、需求裝備、攻略方式（有些侵獸的攻略非常長，像侵獸千刃龍有完整的回合攻略跟咆哮判斷表）、解鎖的危惧種魔物名稱
- 每隻侵獸的攻略方式都大幅擴充了，從原本的一兩句話變成完整的戰鬥攻略指南（全部 9 隻侵獸都有）
- 被侵獸解鎖的危惧種魔物頁面，會顯示「侵獸來源」區塊，告訴你是哪隻侵獸解鎖的、在哪個地區

**突然變異系統**

- 可以從派生來源的魔物（例如火龍）看到牠能變異成哪些物種（亞種、二名），包含變異條件
- 變異後的魔物（例如蒼火龍）頁面會顯示「變異來源」，告訴你是從誰變來的、需要什麼條件
- 侵獸頁面也能看到「可派生突變物種」，顯示亞種和二名的派生條件

**天變古龍攻略**

- 新增 5 隻天變古龍的專屬攻略區塊（天變溟波龍、天變風神龍、天變雷神龍、天變浮岳龍、天變冰咒龍），包含出現地區、地點、攻擊屬性、是否可孵蛋、詳細攻略要點

**侵獸分類**

- 9 隻侵獸（侵獸青熊獸、侵獸角龍、侵獸奇怪龍、侵獸影蜘蛛、侵獸慘爪龍、侵獸水龍、侵獸千刃龍、侵獸鎌蟹、侵獸黑狼鳥）正式獨立為「侵獸」分類，從原本藏在 invasive.ts 的純資料變成可以在列表頁看到、點進去有詳細頁
- 篩選面板的「分類」新增了「侵獸」和「突然變異」兩個選項。選「突然變異」會列出所有跟突然變異有關的魔物（變異來源 + 變異結果都算）

**搜尋功能大幅升級**

- **拼音首字母搜尋**：現在可以打拼音首字母就搜到魔物了，例如打 `hll` 就能找到「火龍」、打 `lyl` 就能找到「雷鬃龍」
- **模糊音搜尋**：zh/z、ch/c、sh/s、n/l、r/l、f/h、ang/an、eng/en、ing/in 這些容易搞混的音都能互相匹配，例如打「火龙」的拼音但用了 `huolong` 也找得到
- **英文模糊搜尋**：英文搜尋現在支援 edit distance（容錯距離），打錯一兩個字也能搜到。例如打 `ratholos`（少了個 a）也能找到 Rathalos
- 搜尋框的 placeholder 更新為「搜尋魔物名稱 / 拼音 / 縮寫 / English...（Ctrl+K）」

### 資料修正與名稱統一

**英文名稱大改版 — Afflicted → Feral**

- 所有兇異魔物的英文名稱從 `Afflicted` 改成 `Feral`，跟遊戲內的英文版統一。這個改動影響了 23 隻兇異魔物，牽動了 monsters.ts、locations.ts、weakness.ts、damageEffectiveness.ts 以及相關魔物家族對照全部的 key 和命名

**中文名稱修正**

- 白電龍 → 奇怪龍（Khezu），紅電龍 → 紅怪龍（Red Khezu），跟遊戲正式翻譯統一
- 血橙獸 → 緋天狗獸（Blood Orange Bishaten），跟遊戲正式翻譯統一
- 侵獸白電龍 → 侵獸奇怪龍，連帶 invasive.ts 裡的資料也一起改了
- 兇異纏蛙的特殊攻擊從「鑛物纏」改成「鑛物纏繞」

**「異常狀態有效性」標題改名**

- 詳細頁的 section title 從「異常狀態有效性」改成「狀態異常耐性」，更準確

**屬性弱點資料補完**

- 修正束縛冰狼龍（Hyper Lunagaron）的弱點：從全部「不明」改為正確數值（火大弱點、冰耐性）
- 修正兇異怨虎龍（Feral Magnamalo）的弱點：從部分「不明」改為完整數值（水大弱點、雷大弱點）
- 修正侵獸黑狼鳥（Invasive Yian Garuga）的弱點：從部分「不明」改為完整數值
- 修正侵獸青熊獸（Invasive Arzuros）的弱點
- 修正侵獸影蜘蛛（Invasive Nerscylla）的弱點
- 修正侵獸水龍（Invasive Plesioth）的弱點
- 修正侵獸慘爪龍（Invasive Odogaron）的弱點
- 修正侵獸角龍（Invasive Diablos）的弱點
- 修正侵獸千刃龍（Invasive Seregios）的弱點
- 修正溟波龍（Namielle）的弱點：補上原本不明的雷耐性和冰弱點
- 修正冰咒龍（Velkhana）的弱點：補上完整數值
- 修正風神龍（Wind Serpent Ibushi）的弱點
- 修正雷神龍（Thunder Serpent Narwa）的弱點

**異常狀態有效性資料補完**

- 修正棘龍（Espinas）：從大部分「❓」改為正確數值（麻痺雙叉、睡眠雙叉、爆破雙星、目眩三星等）
- 修正溟波龍（Namielle）：從部分「❓」改為正確數值（毒三星、火傷雙星）
- 修正侵獸黑狼鳥（Invasive Yian Garuga）：從部分「❓」改為正確數值

### 技術 / UI 改動

- **侵獸 icon 處理**：兇異魔物的 icon override 被註解掉了（因為現在兇異魔物改用 Feral 前綴，有對應的 icon 圖片了），新增了兇異妃蜘蛛、兇異浮岳龍、兇異迅猛龍王（2 種）等新 icon 圖片
- **侵獸 / 突變 / 天變古龍的完整 CSS 樣式**：新增了 `.invasive-info`、`.invasive-detail-grid`、`.invasive-row`、`.evo-card`、`.mutation-info`、`.mutation-target-card`、`.inline-monster-link` 等全新樣式區塊，包含侵獸攻略的 label-value 排版、派生鏈的卡片樣式（亞種跟二名用不同顏色）、行內魔物連結的藍色樣式
- **搜尋引擎重寫**：`search.ts` 加了拼音首字母快取、模糊音正規化表、edit distance 演算法、fuzzyContains 子字串模糊比對，搜尋邏輯從原本的 4 行變成完整的多策略匹配（中文直接→英文直接→日文直接→拼音→中文拼音→模糊音→拼音首字母→英文 fuzzy）
- **分類篩選邏輯更新**：「突然變異」分類不是直接比對 `group`，而是用 `isMutationResult` 和 `isMutationSource` 來判斷，所以能正確篩選出所有跟變異相關的魔物
- **相關魔物家族擴充**：侵獸也被加入了相關魔物家族（例如青熊獸↔侵獸青熊獸、慘爪龍↔侵獸慘爪龍等），方便在詳細頁互相跳轉
- 速龍的外觀 icon 修正：Exotic Velocidrome 從錯誤的 Invasive_Arzuros icon 改回 Velocidrome

## v1.1.0（2026-03-19）

這次更新蠻大的，主要是把魔物詳細頁的資訊補得更完整，列表頁的篩選排序也大幅升級了，另外還加了語音搜尋。

### 新功能

**魔物詳細頁 — 新增好幾個資訊區塊**

- **異常狀態有效性**：現在每隻魔物的頁面都能看到牠對各種異常狀態的抗性了，包含毒、火傷、麻痺、睡眠、爆破、出血、目眩、暈眩、屬性封印、技能封印、龍捲風、即死，總共 12 種。用星星數量來表示有效程度，還有對應的小 icon，一眼就能看懂
- **傷害效果表（部位弱點）**：新增斬擊 / 打擊 / 穿刺對各部位的有效度，用星星（有效）跟叉叉（無效）顯示。像是有些魔物有多種型態（通常、憤怒、帶電等），會有 tab 可以切換看不同狀態下的部位弱點
- **賓果數加成**：顯示每隻魔物 2 賓果 / 3 賓果 / 5 賓果各自的加成效果（像是破龍力、會心率、速度、耐力等），規劃基因的時候很好用
- **相關魔物**：頁面底部會列出同系列的魔物（原種、亞種、二名、兇異等），有小圖示跟名稱，點了就能直接跳過去。總共建了 40 幾組魔物家族的對照關係

**列表頁篩選與排序大升級**

- **地區篩選**：可以用四大區域（亞茲萊爾領域、卡納爾塔密林、塔基爾坎、桑提亞）來篩選，還有「展開子地點」按鈕可以看到更細的 16 個子地點來做精確篩選
- **騎乘能力篩選**：新增 7 種騎乘能力的篩選 chip（飛行、攀爬、游泳、潛地、隱密、咆哮、吐息），可以多選，選了的話就只會顯示具備所有你選的能力的魔物
- **騎乘攻擊屬性篩選**：可以依騎乘攻擊的屬性來篩（火、水、雷、冰、龍、無），選中的 chip 會顯示對應的屬性顏色
- **排序功能**：新增排序下拉選單，有五種排序方式 — 預設排序、中文名稱、英文名稱、種類、地區

**語音搜尋**

- 搜尋欄旁邊多了一個麥克風按鈕，按下去就可以用語音來搜尋魔物名稱。用的是瀏覽器內建的 Web Speech API，語言設定為繁體中文（zh-TW），支援即時辨識（邊講邊顯示）。錄音中按鈕會變紅色並閃爍提示

**基因規劃器改善**

- 在九宮格放上基因之後，下方的基因查詢列表會自動篩選成只顯示你有用到的基因，不用再從一大堆基因裡面慢慢找了。提示文字也會跟著變

### 資料新增與修正

**新增資料**

- 新增 23 隻兇異魔物的出現地點與取得方式資料（兇異查加布魯、兇異毒怪鳥、兇異速龍王、兇異毒怪鳥、兇異泡狐龍、兇異大名鑽蟹……等等）
- 新增異常狀態有效性資料（statusEffectiveness.ts），涵蓋大部分魔物
- 新增傷害效果 / 部位弱點資料（damageEffectiveness.ts），含多型態資料
- 新增賓果加成資料（bingoBonus.ts），涵蓋大部分可騎乘魔物
- 新增全部 7 張異常狀態 icon 圖片（毒、火傷、麻痺、睡眠、爆破、出血、暈眩）
- 新增地區列表資料（4 大區域 + 16 個子地點），支援地區篩選用

**資料修正**

- 修正水獸（Ludroth）的特殊攻擊：從 `-` 改為 `[憤怒]: 技巧`
- 修正恐暴龍（Deviljho）的屬性弱點：從全部「不明」改為正確數值（冰大弱點、龍微耐性）
- 修正 Dos Poogie 的基因資料 key（從 `Dospoogie` 改成 `Dos Poogie`，跟其他地方統一）
- 修正 3 個基因翻譯名稱：蒼雷電壓→青電壓、蒼雷電漿→電漿光束、雷光刃→光刃斬
- 統一「危惧種巢穴」用語為「侵獸巢穴」（影響電龍、雷狼龍、迅龍、轟龍、冰牙龍、雌火龍、電龍等 8 隻魔物的取得方式描述）
- 暫時註解掉尚未確認完整資料的 4 隻古龍種（麒麟、炎王龍、鋼龍、滅盡龍），等資料補齊再放回來
- 新增兇異魔物的弱點對照表（Feral → Feral 對應），讓兇異魔物也能正確顯示屬性弱點

### 技術 / UI 改動

- **魔物 ID 格式改版**：從流水號 `monster-0`, `monster-1` 改成用英文名的 slug（例如 `rathalos`, `azure_rathalos`），網址更好看也更穩定，不會因為資料順序調整就壞掉
- **ScrollToTop 元件**：切換頁面時會自動捲回頂端，之前從詳細頁回列表頁會卡在奇怪的位置
- **TypeScript 型別補充**：加了 `BingoBonus` interface、`SpeechRecognition` 相關的 Web API 型別定義（在 vite-env.d.ts），還有 `SortOption`、`RideAbility` 等新型別
- **篩選面板 UI 重整**：移除了一些多餘的 `filter-header` 包裝 div，排版更乾淨。地區篩選的子地點有左邊線條的縮排效果
- **搜尋欄 padding 調整**：右邊空間加大，讓語音按鈕跟清除按鈕不會擠在一起
- **全新 CSS 樣式**：傷害效果表、異常狀態表、賓果加成、相關魔物卡片、排序選單、地區 chip、騎乘能力 chip、騎乘屬性 chip、語音按鈕動畫，全部都是新做的
- Dos Poogie 的 icon 圖片有更新過（檔案大小從 23KB → 356KB）

- 魔物圖鑑：超過 130 隻魔物的清單，支援中 / 英 / 日搜尋，可用種族、攻擊類型、分類篩選
